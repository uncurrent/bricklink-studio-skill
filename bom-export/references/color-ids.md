## Notes

- BrickLink color IDs differ from LDraw — always use this table when generating BL XML
- Some LDraw colors have no exact BrickLink equivalent — use closest match
- For current/updated mapping, see: https://www.bricklink.com/catalogColors.asp
- Studio uses LDraw color IDs internally
