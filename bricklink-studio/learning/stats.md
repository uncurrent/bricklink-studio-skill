# Session Statistics

Tracks skill usage across sessions. Updated automatically at end of each session.
Used by maintainers to understand which sub-skills are most used and where the skill fails.

Format per entry:
```
## YYYY-MM-DD — Session N
**Sub-skills used:** list of sub-skills
**Observations logged:** N
**Promoted to patterns:** N
**Moved to failed:** N
**Discarded:** N
**User rating:** ⭐⭐⭐⭐⭐ / 🔄 / ⚠️ / ❌
**User note:** (optional)
```
