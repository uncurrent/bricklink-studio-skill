# BrickLink Studio — Keyboard Shortcuts

## Viewport Navigation
| Key | Action |
|---|---|
| Right mouse drag | Rotate camera |
| Middle mouse / Scroll | Zoom |
| Shift + Middle mouse | Pan |
| Numpad 1 | Front view |
| Numpad 3 | Right side view |
| Numpad 7 | Top view |
| Numpad 5 | Toggle perspective / orthographic |
| Numpad 0 | Camera view (if set) |
| F | Frame selected part |
| A | Select all |

## Part Placement (while placing a part from the palette)
| Key | Action |
|---|---|
| R | Rotate the part being placed |
| T | Move the part being placed |

## Editing
| Key | Action |
|---|---|
| W | Move tool |
| E | Rotate tool |
| G | Snap to grid toggle |
| B | Open Parts palette |
| C | Open Color palette |
| Delete | Delete selected part(s) |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl+D | Duplicate selected |
| Ctrl+G | Group selected |
| Escape | Deselect / cancel action |

## File
| Key | Action |
|---|---|
| Ctrl+S | Save |
| Ctrl+Shift+S | Save As |
| Ctrl+N | New model |
| Ctrl+O | Open file |
| Ctrl+E | Export (LDraw) |

## Rendering & View
| Key | Action |
|---|---|
| F11 | Start render |
| H | Hide selected part(s) |
| Shift+H | Show all hidden |
| 1–9 | Visibility layers |

## Instructions Mode
| Key | Action |
|---|---|
| Ctrl+Right | Next step |
| Ctrl+Left | Previous step |
| Ctrl+Shift+N | Add new step |
