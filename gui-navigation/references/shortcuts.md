## Building Mode

### Brick Control

| Result | Windows | Mac |
|---|---|---|
| Force Release Brick | Space | Space |
| Escape / Deselect | Esc | Esc |
| Select All | Ctrl+A | Cmd+A |
| Move Left | A | A |
| Move Right | D | D |
| Move Forward | W | W |
| Move Backward | S | S |
| Rotate Forward 90° | Up Arrow | Up Arrow |
| Rotate Backward 90° | Down Arrow | Down Arrow |
| Rotate Left 90° | Left Arrow | Left Arrow |
| Rotate Right 90° | Right Arrow | Right Arrow |
| Rotate Forward 45° | Shift+Up | Shift+Up |
| Rotate Backward 45° | Shift+Down | Shift+Down |
| Rotate Left 45° | Shift+Left | Shift+Left |
| Rotate Right 45° | Shift+Right | Shift+Right |
| Translation ActionPalette | T | T |
| Rotation ActionPalette | R | R |
| Split | Alt+S | — |
| Merge | Alt+M | — |

**WASD vs arrows:** WASD moves part freely without snapping and without collision checks — useful for rough positioning. Arrows rotate the part 90° (or 45° with Shift). Use WASD to get close, then arrows to snap precisely.

**Translation ActionPalette (T):** the yellow square resets the movement axis orientation — useful when parts inside a rotated group move in unexpected directions.

**Focus (F):** select a part and press F — camera locks onto it and all orbit/zoom centers on that part. To reset: right-click → Reset Origin.

## Precision Placement

Right-click selected part → **Properties** → enter exact X, Y, Z coordinates and rotation matrix values.
Useful when pixel-perfect positioning is required or nudging is too coarse.
