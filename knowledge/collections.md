## COL01: Awesome LEGO Machine Learning

**Summary:** The primary curated index of all ML/AI/algorithmic projects related to LEGO.
Updated regularly. Start here for any LEGO + ML question.
**Source:** https://github.com/360er0/awesome-lego-machine-learning
**Tags:** curated list, ML, entry point, comprehensive

### Categories covered in the list

| Category | Examples already catalogued |
|---|---|
| Recognition & Classification | → part-recognition.md |
| Assembly & Design | → assembly-planning.md |
| Generation & Conversion | → model-generation.md |
| Rendering & Datasets | → datasets-rendering.md |
| Sorting Machines | Not yet in domain files — see list directly |

### Not yet in domain files (sorting machines)

Notable projects from the list worth knowing:
- **Universal LEGO Sorting Machine** (2022) — conveyor belt + DL, built for a BrickLink store
- **Deep Learning Lego Sorter** (2021) — DIY sorter built from LEGO, with public code/instructions
- **DISBY** (2021) — first automated system using minimal formal descriptions (size, weight) for recognition

## COL04: Awesome LEGO (ad-si)
